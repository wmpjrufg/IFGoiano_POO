O projeto final

Este projeto foi desenvolvido ao longo do meu treinamento na plataforma Alura.

O primeiro passo deste projeto consiste no entendimento do elemento principal de um banco a conta corrente. Vamos entender quais os dados ou atributos de uma conta corrente.

  1. Saldo
  2. Agência
  3. Número
  4. Titular

Quais operações podem ser feitas por exemplo dentro de uma conta:

  1. Saque
  2. Transferência
  3. Depósito

Para iniciar o algoritmo do IFBank vamos criar o arquivo auxiliar, Conta.java, que guardará as informações da classe conta. Para isso criaremos o seguinte código dentro deste arquivo .java

```java class Conta { double saldo; int agencia; int numero; String titular; } ```

Para criar uma nova conta (a primeira conta) utilziando a linguagem Java devemos empregar a palavra chave new. Portanto escreveremos o seguinte código:

```java public class CriaConta { public static void main(String[] args) { Conta joaoConta = new Conta(); System.out.println(joaoConta); } } ``` ``` Output Conta@76ed5528 ```

Perceba que o código Java no seu output referência um endereço onde está guardada a informação do objetvo instânciado. Além disso é possível observar a variável joaoConta é do tipo Conta. Tal fato faz referência a classe criada no arquivo Conta.java.

Agora vamos estabelecer um saldo para esta conta e imprimir no console o mesmo:

```java public class CriaConta { public static void main(String[] args) { Conta joaoConta = new Conta(); joaoConta.saldo = 2000; System.out.println("Saldo joão: $ " + joaoConta.saldo); } } ``` ``` Output Saldo joão: $ 2000.0 ```

Preenchendo os atributos do primeiro objeto:

```java public class CriaConta { public static void main(String[] args) { Conta joaoConta = new Conta(); joaoConta.saldo = 2000; System.out.println("Saldo joão: $ " + joaoConta.saldo); } } ``` ``` Output Saldo ```
